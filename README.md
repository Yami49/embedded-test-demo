# Embedded Test Demo

Small demo project simulating an embedded temperature controller with automated tests.

## Tech Stack
- C++
- Google Test
- Python (test runner)

## Features
- Simple control logic (heater on/off)
- Unit tests for validation
- Automated build & test execution

python3 run_tests.py
